# 시작버튼에서 Git Bash 열기(실행)

```
git config --global user.name "gowoon"
```

# github 가입시 입력한 이메일과 동일해야한다.
```

```

# GitHub에 코드 업로드하기 💛

1. 초기화
  
  ```
  git init
  ```

# .git이라는 폴더가 생성된다

2. 파일 올리기

   ```
   git add .
   ```

3. history 만들기
   ```
   git commit -m "내가 적고싶은 메세지"
   ```
   ☑️ 메세지에는 한글이 가능함
